# Overview

**PSKracker** is a collection of WPA/WPA2 default algorithms/password generators written in C.

# Requirements

```
None
```

# Setup
**Download**

`git clone https://github.com/soxrok2212/PSKracker`

**Build**

cd PSKracker
gcc att.c -o att
